Tables

Every studentwho's being forced to learn html4's worst nightmare.
HTML tables are easy to use and make quickly. The problem arises when you are being tested on them.
I will attempt my best to make them easy to understand. I will be using basic html tables but there isn't much difference in moving the tr tags into a tbody tag.

Rowspan

Here is an example table using rowspan.
Name Age
Ajay 24
Priya
As you can see, setting the colspan makes that cell merge to columns. while this seems intuitive enough to be remembered at the first glance you may get confused with colspan when asked about it's function. It's best to create a strong assocition of vertical merging and the word rowspan if English isn't your mother tongue.
Do not add excessive td tags in rows which are being spanned by the rows above The number of td tags a row should contain are table's rows - rows spanned by rows above
Name Age
Ajay 24
Priya Adding a second td tag in the spanned row is gonna break the table
